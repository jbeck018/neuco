## Competitive Positioning (as of 2026-03-12)

**Core moat:** No competitor does signal→spec→code. Neuco is the only platform connecting customer feedback directly to shipped code via structured specs + Storybook stories + GitHub PRs.

**Signal/Insight tools (stop at aggregation):**
- Productboard ($19-59/maker/mo + $20 AI add-on) — feedback aggregation, PRD gen via Spark AI. No code.
- Dovetail ($15/user/mo) — research analysis, transcription. No code.
- Enterpret (custom pricing) — 50+ source aggregation, anomaly detection. Enterprise-only.
- Unwrap.ai (Series A 2025) — auto-tagging, NL querying. Early stage.

**Code gen tools (require manual prompts):**
- v0 by Vercel ($20/mo) — React/Next.js UI gen. No signal ingestion.
- Lovable ($25-50/mo) — full-stack MVPs. No signal ingestion.
- Bolt ($20-200/mo) — browser-based full-stack. No signal ingestion.
- Cursor ($20/mo Pro) — AI code editor. Manual context.
- GitHub Copilot Workspace (GA 2026) — plan-based codegen from Issues.

**TAM:** ~$800M-$1.2B (intersection of product intelligence + AI code gen)

**Pricing validation:** $49-399/mo tiers competitive. Undercuts Productboard + v0 bundle.

**Threats:** Productboard code gen (12-18 mo), Copilot Workspace expansion (18 mo).

**Why:** Informs roadmap priorities — focus on synthesis quality and GitHub integration depth as defensible moats.
**How to apply:** When evaluating feature priorities, weight things that deepen the signal→code pipeline over point features that competitors already do well.
